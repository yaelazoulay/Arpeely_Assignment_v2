**Metrics rationale (Task 2)**

We focused on metrics that reconstruct the full journey available in the extract—**LP interaction → install → in-product trial**—without relying on fields that are not in the file (e.g. ad impressions for true CTR). **Click and install volumes**, **install CVR by user and by landing page (first-touch attribution)**, and **daily event trends** answer whether acquisition is healthy and which campaigns earn their traffic. **Post-install trials** (`try_grammarly` by `product_feature`) and **adoption rate** distinguish “installs that engage” from hollow top-of-funnel wins; **time from first click to install** and the **click-without-install** cohort support retargeting and intent segmentation.

Together, these KPIs measure campaign success as **efficient conversion plus quality**: strong LPs raise attributed CVR, strong product messaging raises trial adoption after install, and stable daily install counts with predictable funnel ratios indicate sustainable performance rather than one-off spikes. The four dashboard panels map directly to **volume monitoring**, **LP benchmarking**, **feature engagement**, and **drop-off diagnosis**—the actions a growth team would take within a day of seeing the data.
